# [STRESSED] — Game Concept

เกมจะเป็นเนื้อเรื่องเกี่ยวกับพิธีกรรมเรียกสิ่งโบราณขึ้นมา เกมจะถูกเซทไว้ในช่วงสงครามโลกครั้งที่สองของช่วงท้ายสงคราม ณ ขณะนั้นมีกลุ่มลัทธิคลั่งที่สนับสนุนโดยรัฐบาลนาซีเอง วันนึงได้ทำพิธีกรรมเรียกปลุกสิ่งโบราณขึ้นมา แต่เกิดเหตุผิดพลาด จนทำให้พิธีกรรมแตก เราคือหนึ่งในผู้ที่พยายามหนีจากหายนะครั้งนี้

เกวางแผนฌฟทำจะเป็นแนวsurvivol hororผู้เล่นต้องหลบ หนี ซ้อน และรอดไปให้ได้ระหว่างการหนีผู้เล่นจะได้เจอผู้รอดชีวิตคนอื่น แต่พวกเขาอาจอยู่กับเราได้ไม่นานมากเป็นแค่ททางผ่าน

## Elevator Pitch

[อธิบายเกมใน 2-3 ประโยค: เกมอะไร, ผู้เล่นทำอะไร, ทำไมถึงสนุก]

เกมแนวระทึกขวัญสิ่งลึกลับ ต้องซ้อนหนีหลบ ผู้เล่นได้คิดได้แก้ปัญหาในการเอาชีวิตรอด
